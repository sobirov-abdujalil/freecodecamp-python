# freeCodeCamp Python Curriculum

This repository follows the official **freeCodeCamp Python Certification (v9)** curriculum.

## Curriculum Structure

The certification is organized into modules covering Python fundamentals through advanced topics:

| # | Module | Topics |
|---|--------|--------|
| 01 | **Python Basics** | Introduction, variables, strings, numbers, conditionals, functions |
| 02 | **Python Installation** | Setting up Python locally |
| 03 | **Loops and Sequences** | `for`, `while`, lists, tuples, range |
| 04 | **Dictionaries and Sets** | Dicts, sets, modules, imports |
| 05 | **User Configuration Manager** | Certification project |
| 06 | **Error Handling** | `try`/`except`, debugging |
| 07 | **Classes and Objects** | OOP fundamentals |
| 08 | **Budget App** | Certification project |
| 09 | **Object-Oriented Programming** | Encapsulation, inheritance, polymorphism, abstraction |
| 10 | **Polygon Area Calculator** | Certification project |
| 11 | **Data Structures** | Stacks, queues, linked lists |
| 12 | **Hash Table** | Certification project |
| 13 | **Algorithms** | Binary search, merge sort, quicksort, Luhn algorithm |
| 14 | **Tower of Hanoi** | Certification project |
| 15 | **Graphs and Trees** | BFS, DFS, shortest path, N-Queens |
| 16 | **Dynamic Programming** | Fibonacci, optimization |
| 17 | **Review and Exam** | Certification exam prep |

Each folder contains starter files with example code for the corresponding freeCodeCamp lessons, workshops, and labs.

## Official Curriculum

Start learning at: [freecodecamp.org/learn/python-v9](https://www.freecodecamp.org/learn/python-v9)

## How to Use

1. Complete the lessons, workshops, and labs on freeCodeCamp
2. Use the files here as reference or starting points
3. Add your own solutions and experiments in each module folder
4. Practice by extending the examples with your own ideas
